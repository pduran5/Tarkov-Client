===============================================
           Uso del Cliente Tarkov v0.1.5
===============================================

Modificación del Cliente Tarkov v0.1.5 basado en el proyecto original de TarkovClient.
Aplicación de escritorio dedicada para Tarkov Market Pilot

[Introducción al Programa]
Este programa ejecuta el sitio web Tarkov Market Pilot como una aplicación de escritorio,
proporcionando detección de mapa en tiempo real y funciones de seguimiento de ubicación en integración con el juego.
El sitio web Tarkov Market se cachea en la carpeta "Cache" para que el programa pueda funcionar sin conexión.

===============================================
                Cómo Ejecutar
===============================================

1. Haz doble clic en el archivo TarkovClient.exe
2. El programa comenzará automáticamente
3. Aparecerá un icono en la bandeja del sistema (abajo a la derecha)

※ ¡No se requiere proceso de instalación separado!

===============================================
            Advertencia de Windows Defender
===============================================

Puede aparecer una advertencia de seguridad al ejecutar por primera vez.
Esto se debe a que no se ha comprado un certificado de firma de código,
no hay problemas de seguridad con el programa en sí.

[Cómo Resolver]
1. Haz clic en "Más información"
2. Haz clic en el botón "Ejecutar de todas formas"
3. El programa se iniciará normalmente

※ Este programa es de código abierto y todo el código es público y seguro.

===============================================
                Uso Básico
===============================================

[Gestión de Pestañas]
- Añadir nueva pestaña: Haz clic en el botón + en la parte superior izquierda
- Cerrar pestaña: Haz clic en el botón X de cada pestaña
- Siempre se mantiene al menos 1 pestaña

[Integración con el Juego]
- La detección de mapa comienza automáticamente cuando ejecutas el juego
- La ubicación se actualiza cuando tomas una captura de pantalla en el juego
- Los archivos de registro se limpian automáticamente cuando se cierra el juego

===============================================
               Uso del Modo PiP
===============================================

[¿Qué es el Modo PiP?]
Es una función que cambia automáticamente a una pequeña ventana de minimapa cuando cambia el mapa durante el juego.
Es muy conveniente porque puedes verificar el mapa incluso durante el juego.

[Cómo Configurar PiP]
1. Haz clic en el botón de configuración ⚙️ en la barra lateral izquierda
2. Marca "Activar modo PiP globalmente"
3. Configura la activación/desactivación individual para cada mapa deseado
4. Haz clic en el botón "Guardar"

[Uso de Teclas Rápidas PiP]
1. Marca "Usar botón de activación PiP" en la configuración
2. Establece la tecla rápida deseada (Valor predeterminado: F11)
3. Minimiza la pantalla PiP en cualquier momento durante el juego con la tecla rápida

[Controles del Modo PiP]
- Mover ventana: Arrastra la ventana del minimapa a la posición deseada
- Cerrar ventana: Haz clic en el botón X en la esquina superior derecha
- Recordar posición: Guarda automáticamente la posición con la opción "Guardar UI de pantalla PiP más reciente"

[Mapas Soportados]
✓ Factory (Fábrica) ✓ Woods (Bosques) ✓ Customs (Aduanas) ✓ Reserve (Reserva)
✓ Ground Zero ✓ Streets of Tarkov ✓ Lighthouse (Faro)
✓ Interchange (Intercambio) ✓ Shoreline (Costa) ✓ Labs (Laboratorios)

※ Se configura automáticamente con el tamaño y posición optimizados para cada mapa

===============================================
               Solución de Problemas
===============================================

[Si el programa no se inicia]
- Intenta ejecutarlo como administrador
- Actualiza Windows a la última versión

[Si la página web no carga]
- Verifica tu conexión a internet
- Permite el programa en tu firewall

[Si no se detecta el mapa]
- Verifica si el juego está instalado en la ubicación predeterminada
  (C:\Battlestate Games\Escape from Tarkov\)
- Intenta ejecutar el juego como administrador

[Si el modo PiP no funciona]
- Verifica si "Activar modo PiP globalmente" está marcado en la configuración
- Verifica si la configuración individual para ese mapa está activada
- Verifica si tienes permisos de acceso a la carpeta de registros del juego

===============================================
              Requisitos del Sistema
===============================================

- Windows 10 o Windows 11 (64 bits)
- Memoria: Mínimo 512MB de espacio libre
- WebView2 Runtime (Incluido por defecto en Windows 11)

[Si se requiere la instalación de WebView2 Runtime]
Los usuarios de Windows 10 deben ejecutar Windows Update.
Si el problema persiste después de la actualización, busca y descarga
'WebView2 Runtime' en el sitio web de Microsoft.

===============================================

===============================================

✓ Detección de mapa en tiempo real - Detección automática de cambio de mapa basada en registros del juego
✓ Rastreo de ubicación - Actualizaciones de ubicación y dirección en tiempo real basadas en capturas de pantalla

✓ Limpieza automática de archivos - Limpieza automática de carpetas de registro y capturas de pantalla (optimización de rendimiento)
✓ Bandeja del sistema - Se ejecuta silenciosamente en segundo plano
✓ Cacheo de los mapas en local - Cacheo de los mapas en local para que el programa pueda funcionar sin conexión.
